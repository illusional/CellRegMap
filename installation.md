## Install

From your command line, enter

    pip install cellregmap

in your command line.

## Development

To install it in development mode, enter

    git clone https://github.com/limix/CellRegMap.git
    cd CellRegMap
    pip install -e .

in your command line.

## Running tests

From your command line, enter

    python setup.py test
